# Full Module Phase Checklist

Use this checklist for every full module build.

## Before Build

- Confirm module name
- Confirm route path
- Confirm API route path
- Confirm tier access
- Confirm usage limits
- Confirm tutorial content

## During Build

- Add/replace all module files
- Add dashboard page
- Add API routes
- Add config
- Add tutorial
- Add usage rules
- Add access logic

## After Build

```bash
npm run build
npm run dev
